Scrollbars for MooTools 1.4

This little script makes it easy to add custom scrollbars to almost any HTML element on your website in any (modern browser).

Implementation

Implementing the scrollbars script is pretty straightforward. First include the MooTools library and scrollbars.min.js in your head.

Adding one of the pre-designed examples is optional. 

<head>
	<script src="mootools/mootools-core-1.4.5.js"></script>
	<script src="scrollbars/scrollbars.min.js"></script>
	<!-- Optional pre-designed -->
	<link href="examples/osx/style.css" rel="stylesheet" />
	<script src="examples/osx/script.js"></script>
</head>

If you want to style your own scrollbars you just select the element(s) that you want to add custom scrollbars to and run the .scrollbars() method on them. By default the scrollbars have minimal style, check out the structure to know how and which classes can be used to style the bars.

Check out the options table to see which settings can be set/changed. These options can be passed as object to the scrollbars method; .scrollbars({option: value})

<script>
window.addEvent('domready',function(){
	$$('.your-wrapper').scrollbars();
});
</script>

Options

Option Default value Description
scrollBarSize 15 Size of the scrollbar elements (buttons and bar width).
scrollStep 20 Steps in pixels it takes when buttons are pressed.
addBar true false disables the bars, which makes the scrollbars only usable by buttons that are generated.
barOverContent false true places the scrollbars over the content (like OSX Lion and iOS).
fade false Can either be set to true or to the milliseconds it should take till fading out.

HTML structure

This is the structure that will be build to create the custom scrollbars. There will also be some default styles added to the elements, it depends on the situation which that will be. You might need to add !important to your style properties to override those settings.

We suggest to inspect our example files to see how certain styles are implemented. 

<div class="your-wrapper">
	<ul class="scrollbar horizontal">
		<li class="scroll bar-wrapper">
			<div class="scroll bar"></div>
		</li>
		<li class="scroll left">left</li>
		<li class="scroll right">right</li>
	</ul>
	<ul class="scrollbar vertical">
		<li class="scroll bar-wrapper">
			<div class="scroll bar"></div>
		</li>
		<li class="scroll up">up</li>
		<li class="scroll down">down</li>
	</ul>
	<div class="scrollbar-content">
		<div class="scrollbar-content-wrapper">
			<!-- Original content from <div class="your-wrapper"> -->
		</div>
	</div>
</div>
